### Dart
    Dart是单线程模型，通过消息循环来执行任务[类似node.js]

dart中所有的东西都是对象。

### 内置数据类型
- Numbers
- Strings
- Booleans
- Lists
- Maps

"??" 数据域操作符 
eg：AA ?? "999" 表示如果AA为空，返回999
eg: AA ??= "999" 表示如果AA为空，给AA赋值999

"_" 代表private


dart 中在没有显示的指定变量的类型时，debug模式下变量类型都是dynamic类型

使用".."级联符号可以在一个对象上进行一系列的操作。

如果函数只有一条语句，可以使用"=>"
